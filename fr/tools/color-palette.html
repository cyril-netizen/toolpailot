Questions Fréquemment Posées

Qu'est-ce qu'une palette de couleurs harmonieuse?

Une palette harmonieuse est un ensemble de couleurs qui fonctionnent bien ensemble. Elle peut être basée sur la théorie des couleurs, utilisant des relations comme complémentaire, triadique ou analogique pour créer une cohésion visuelle.

Quelle est la différence entre les modes de couleurs?

Aléatoire crée des couleurs au hasard. Complémentaire utilise des couleurs opposées. Triadique utilise trois couleurs équidistantes. Analogique utilise des couleurs proches les unes des autres.

Puis-je exporter la palette?

Oui, vous pouvez copier le code CSS directement et l'utiliser dans vos projets. Vous pouvez aussi copier chaque couleur individuellement en cliquant sur la carte de couleur.

Questions Fréquemment Posées

Qu'est-ce qu'un slug URL?

Un slug URL est la partie lisible d'une URL qui décrit le contenu de la page. Par exemple, dans "blog.com/mon-super-article", "mon-super-article" est le slug. Les slugs doivent être courts, descriptifs et optimisés pour le SEO.

Pourquoi supprimer les accents?

Supprimer les accents rend les URLs plus compatibles avec tous les systèmes et navigateurs. Une URL sans accents est aussi plus lisible et facile à partager. Par exemple, "été" devient "ete".

Puis-je utiliser un séparateur personnalisé?

Oui, vous pouvez utiliser n'importe quel séparateur. Le trait d'union (-) est le plus courant et recommandé pour le SEO, mais vous pouvez aussi utiliser un tiret bas (_) ou un point (.).

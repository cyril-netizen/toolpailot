Questions Fréquemment Posées

Qu'est-ce que Markdown?

Markdown est un langage de balisage léger qui permet de formater du texte de manière simple et lisible. Il est largement utilisé pour les documentations, les blogs et les fichiers README.

Quels éléments Markdown sont supportés?

Notre aperçu supporte les titres, le texte gras et italique, les listes, les blocs de code, les citations, les liens et les images.

Puis-je exporter le HTML généré?

Oui, vous pouvez copier le HTML généré directement avec le bouton "Copier le HTML" et l'utiliser dans vos projets web.

À Propos de l'Aperçu Markdown

Markdown est devenu la lingua franca de la rédaction technique. Des développeurs aux rédacteurs de documentation, tout le monde utilise Markdown pour sa simplicité et sa lisibilité. Notre aperçu Markdown vous permet de voir instantanément comment votre texte sera rendu en HTML.

Avec notre outil, vous pouvez écrire du Markdown dans le volet de gauche et voir l'aperçu HTML correspondant dans le volet de droite en temps réel. Cela vous aide à formater votre contenu correctement avant de le publier.

Que vous écriviez une documentation, un article de blog ou un README pour un projet GitHub, cet outil vous aide à maîtriser Markdown et à créer du contenu bien formaté.

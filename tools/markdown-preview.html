Markdown Syntax Guide

Headings: Use # for h1, ## for h2, ### for h3, and so on. Markdown automatically converts these to proper HTML heading tags.

Emphasis: Surround text with *asterisks* or _underscores_ for italic, and **double asterisks** or __double underscores__ for bold formatting.

Links: Create links using [link text](URL) syntax. Images use ![alt text](image-url) format with an exclamation mark at the start.

Lists: Use - or * for unordered lists and numbers for ordered lists. Code blocks are created with triple backticks.

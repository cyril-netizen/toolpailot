### Description courte (1 ligne)
Free, privacy-first online tools — JSON formatter, password generator, word counter & more. 100% client-side, no signup.

### Description moyenne (2-3 lignes)
ToolPilot is a collection of free online tools that run entirely in your browser. Format JSON, generate passwords, count words, encode Base64, convert text cases, and more — with zero data sent to any server. Fast, private, no signup required.

### Description longue (5+ lignes)
ToolPilot offers a growing suite of free online utilities for developers, writers, and designers. Every tool runs 100% client-side in your browser — your data never leaves your device.

Available tools: JSON Formatter & Validator, Password Generator, Word & Character Counter, Base64 Encoder/Decoder, Case Converter, Lorem Ipsum Generator, Color Palette Generator, and UUID Generator.

Built with performance and privacy in mind: no accounts, no tracking, no server-side processing. Just open the tool and use it instantly.

### Tags / Keywords
online tools, json formatter, password generator, word counter, base64 converter, developer tools, free tools, privacy, client-side, no signup
